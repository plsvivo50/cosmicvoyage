## 一、资源系统（矿物与原材料）

### 1.1 设计原则

- **原版资源为主**：铁、金、铜、红石、青金石、钻石、下界合金——飞船建造主要消耗这些
- **行星特产为引导**：每个行星有1-2种独特资源，给玩家"去那里的理由"
- **采集方式多样**：地表拾取、浅层采矿、深层钻探、大气提取
- **无稀有度碾压**：特产资源不是"更好的铁"，而是"做特定东西必需的材料"

### 1.2 设计核心：地球全覆盖，外星球高效富集

> **地球上所有资源都存在，但储量极低、开采极慢。**
> **外星球是同种资源的高效富集区。**
>
> 玩家可以选择在地球费力地攒资源造飞船，也可以造一艘基础飞船飞出去高效采集。信息差（"月球有大量钛矿"）驱动探索，而不是资源锁死。

### 1.3 资源分级

| 级别 | 分布 | 说明 |
|:---|:---|:---|
| **基础资源** | 地球丰富 | 铁、金、铜、红石、钻石——原版已有 |
| **普通太空资源** | 地球稀少 / 外星球丰富 | 陨铁、钛矿、氦-3、钨矿——地球上能找到矿脉，但储量低 |
| **高级太空资源** | 地球极稀有 / 外星球丰富 | 云母、托林、深冰核——地球实验室级微量存在 |
| **极限资源** | 地球实验室级微量 / 特定星球富集 | 未知晶体、生物荧光素——地球有痕量，特定星球富集 |

## 二、物品系统

### 2.1 设计原则

- **工具类物品少而精**：每个都有明确不可替代的用途
- **消耗品直观**：氧气瓶、燃料棒、修理包——看一眼就知道干嘛的
- **收集品有叙事重量**：每个样本都是一段发现

### 2.2 工具

| 物品 | 来源 | 用途 |
|:---|:---|:---|
| **宇航服** | 合成（初始可造） | 舱外活动、短时间太空行走 |
| **采样器** | 合成 | 采集生物/地质样本（不破坏目标） |
| **手持扫描仪** | 合成 | 分析地形、发现遗迹信号 |
| **焊接工具** | 合成 | 修复船体裂纹、建造/拆卸飞船方块 |
| **锚定枪** | 合成 | 在小行星/飞船表面固定自己 |

### 2.3 消耗品

| 物品 | 来源 | 用途 |
|:---|:---|:---|
| **氧气瓶** | 基地/飞船补充 | 舱外活动氧气储备 |
| **燃料棒** | 化学/核燃料合成 | 引擎推进剂 |
| **修理包** | 陨铁+硅砂合成 | 修复船体裂纹、模块损伤 |
| **口粮** | 原版食物改造 | 太空环境下的长期储存食物 |

### 2.4 收集品与叙事物品

| 物品 | 来源 | 用途 |
|:---|:---|:---|
| **地质样本** | 各行星采样 | 放入实验室分析、叙事记录 |
| **生物样本** | 木卫二/土卫六 | 放入实验室分析、叙事记录 |
| **前人日志** | 遗迹中找到 | 阅读背景故事 |
| **蓝图残页** | 遗迹中找到 | 查看前人飞船设计（参考用，非解锁） |
| **星文拓片** | 遗迹几何图案 | 收集解读、成就系统 |

## 三、方块系统

### 3.1 设计原则

- **船体方块是核心**：玩家花最多时间与之交互
- **功能方块少但每种都有深度**：不是装饰，是系统
- **外观区别大**：一眼能看出是什么材料/功能

### 3.2 船体方块（飞船建造）

| 方块 | 材料 | 特性 | 用途 |
|:---|:---|:---|:---|
| **标准船壳** | 铁+陨铁 | 基础强度、气密 | 通用外壳 |
| **轻量船壳** | 钛合金 | 同样强度、更轻 | 大型飞船减重 |
| **耐热装甲** | 钨+云母 | 抗高温、抗烧蚀 | 靠近太阳/水星 |
| **压力壳体** | 钢+托林 | 抗高压 | 气态巨行星大气 |
| **抗冻层板** | 固态氮+聚合物 | 抗极寒 | 冥王星深处 |
| **透明舱盖** | 硅砂 | 可看透、弱防护 | 窗户、观测舱 |
| **结构框架** | 铁 | 不提供气密、承重连接 | 内部骨架、扩展连接 |

### 3.3 功能方块（飞船/基地共用）

| 方块 | 功能 | 需求 |
|:---|:---|:---|
| **驾驶舱** | 飞船控制、HUD显示 | 任意飞船必须有且仅有1个 |
| **引擎** | 提供推力 | 至少1个才能起飞 |
| **氧气生成器** | 消耗冰→产生氧气 | 飞船/基地必须有 |
| **太阳能板** | 白天产生电力 | 基地供电、飞船辅助 |
| **核聚变电池** | 持续稳定供电 | 氦-3燃料、高级能源 |
| **气闸门** | 过渡舱、保持气密 | 飞船/基地进出口 |
| **储物舱** | 扩展存储空间 | 无上限、鼓励建造大货船 |
| **实验室** | 分析样本、读取日志 | 样本→知识→记录 |
| **休眠舱** | 跳过时间、保存位置 | 单人存档点、多人同步 |
| **对接端口** | 两艘飞船连接 | 物资转移、人员往来 |
| **钻探模块** | 深层资源采集 | 需飞船着陆目标行星 |
| **大气提取器** | 从大气中提取资源 | 需有大气层的行星 |

### 3.4 基地专用方块

| 方块 | 功能 |
|:---|:---|
| **基础框架** | 放置后自动生成墙壁/地板 |
| **生命维持主机** | 范围供氧、温控 |
| **通讯天线** | 行星间通讯、接收信号 |
| **观测窗** | 大落地窗、可看外面 |

### 3.5 装饰方块

| 方块 | 来源 |
|:---|:---|
| **各行星土壤** | 对应行星采集 |
| **陨石块** | 陨铁合成 |
| **星图墙饰** | 遗迹找到蓝图后合成 |
| **前人纪念碑** | 遗迹中找到 |

## 四、合成系统

### 4.1 设计原则

- **复杂度：中等偏轻**。比原版复杂（需要专用工作台），但不做格雷科技式的复杂产线
- **所有飞船模块一开始就可合成**，无科技锁
- **效率是门槛，不是存在性**：地球上什么都有，只是数量极少、开采极慢。去外星球是为了**高效采集**，不是因为地球没有
- **信息驱动**：遗迹告诉你"月球钛矿储量是地球的千倍"→你去月球→三天采够钛→造轻量船壳。地球也能攒，但可能需要三周
- **资源需求量很大**：一艘中型飞船需要的钛矿，在地球可能要挖几十个废弃矿洞才能凑齐

### 4.2 合成台

| 工作台 | 用途 | 备注 |
|:---|:---|:---|
| **原版工作台** | 基础工具、简单物品 | 起步阶段 |
| **航天装配台** | 飞船方块、功能模块 | 铁+硅砂合成，造了就能用 |
| **精密加工台** | 高级电子元件、跃迁核心 | 需要特产资源 |
| **化学合成台** | 燃料、冷却剂、聚合物 | 涉及化学反应的物品 |

### 4.3 合成复杂度分级

```
Level 1（原版级）：铁+木棍=工具，直接在原版工作台
Level 2（航天级）：陨铁+硅砂=太阳能板，需要航天装配台
Level 3（特产级）：钨+云母=隔热层，需要去特定行星采集
Level 4（稀有级）：未知晶体+氦-3=跃迁核心，多行星资源组合
```

### 4.4 合成设计示例（不做具体网格）

| 产物 | 复杂度 | 关键材料 | 最优采集地 | 地球可行性 |
|:---|:---:|:---|:---|:---|
| 标准船壳 | L1 | 铁+陨铁 | 地球+任意小行星 | ✅ 容易 |
| 太阳能板 | L2 | 硅砂+铜+玻璃 | 地球为主 | ✅ 容易 |
| 隔热层 | L3 | 云母+钨 | 金星+水星 | ⚠️ 极慢（地球钨极少） |
| 轻量船壳 | L3 | 钛合金 | 月球 | ⚠️ 很慢（地球钛稀少） |
| 跃迁引擎 | L4 | 跃迁核心(未知晶体或人造)+氦-3+高级电路 | 冥王星+月球 | ⚠️ 极慢（可用钻石+下界合金+红石在地球合成人造跃迁核心） |
| 宇航服 | L1 | 皮革+铁+玻璃 | 地球 | ✅ 容易 |
| 核聚变电池 | L3 | 氦-3+钨+红石 | 月球+水星 | ⚠️ 极慢（氦-3地球极稀少） |

**跃迁引擎在地球上也能制造**，但需要大量高级原版材料（钻石+下界合金+红石）合成"人造跃迁核心"，耗时极长。去冥王星直接采集未知晶体是高效路径，但不是唯一路径。

## 五、实体系统

### 5.1 设计原则

- **生物极少**：木卫二深海生物、土卫六微生物——其他地方没有生命
- **NPC是前人的影子**：不活着出现，只通过遗迹、日志、蓝图存在
- **无战斗向实体**：没有太空海盗、没有外星怪物
- **环境是敌人**：危险来自物理（引力、辐射、温度），不是生物

### 5.2 生物（有生命实体）

| 生物 | 位置 | 行为 | 交互 |
|:---|:---|:---|:---|
| **木卫二深海生物** | 木卫二冰下海洋 | 缓慢游动、发光 | 可观察、可采样（不伤害） |
| **土卫六微生物群** | 土卫六甲烷湖底 | 显微镜级别、群体发光 | 需实验室分析、可采样 |

**所有生物不攻击、不被攻击、没有血量。它们是发现，不是敌人。**

### 5.3 环境实体

| 实体 | 位置 | 行为 | 危险 |
|:---|:---|:---|:---|
| **小行星** | 小行星带/各行星轨道 | 漂浮、缓慢自转 | 碰撞损伤 |
| **土星环碎片** | 土星环区域 | 高速环绕土星 | 撞击造成结构损伤 |
| **太阳风粒子** | 太阳附近 | 视觉特效 | 辐射盾不足时电子设备故障 |
| **火山喷发物** | 水星/金星/木卫一 | 抛射高温物质 | 碰撞+高温损伤 |

### 5.4 NPC（联动后）

| NPC | 来源 | 功能 |
|:---|:---|:---|
| **宇航员** | MineColonies联动 | 殖民地居民、可分配任务 |
| **科学家** | MineColonies联动 | 实验室工作、样本分析加速 |
| **工程师** | MineColonies联动 | 飞船自动修理、建造加速 |

**本体无NPC。所有单人体验由系统设计完成，不需要NPC填充。**

### 5.5 前人的痕迹（叙事实体）

| 痕迹 | 形式 | 叙事功能 |
|:---|:---|:---|
| **遗骸** | 遗迹中的宇航服/骨骼 | "有人来过这里" |
| **日志终端** | 可交互的损坏设备 | 阅读前人最后记录 |
| **废弃飞船** | 遗迹中的飞船残骸 | 蓝图参考、资源回收 |
| **信息碑** | 刻有星文的石碑 | 星文学习素材 |

## 六、与现有设计的连接

```
资源系统 → 合成系统：外星球资源效率是合成的"时间门槛"
合成系统 → 飞船系统：模块需要特定材料合成
飞船系统 → 探索：新模块=新能力=新地点
探索 → 资源系统：新地点有高效富集的资源

闭环逻辑：
  在地球费力攒资源 → 造基础飞船 → 去月球高效采钛 → 造轻量船壳
      ↓
  新能力（轻量船壳=更大飞船）→ 飞更远 → 发现遗迹获知信息
      ↓
  "金星云母做隔热层可以靠近太阳" → 去金星 → 造隔热层 → 探索水星
      ↓
  效率飞升，文明扩张

关键：每一步都不是"锁死必须去"，而是"去了效率大幅提升"。
玩家始终有选择——没有任何东西被锁死在某个星球上。
```
