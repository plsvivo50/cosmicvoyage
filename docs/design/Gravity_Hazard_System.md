# 引力危险系统设计文档

> 设计哲学：**"宇宙本身就是敌人"**
>
> 没有太空海盗，没有外星军队。真正的危险来自物理本身：引力、辐射、极端环境。

---

## 一、核心概念

```
传统游戏危险："有一个怪物要打"
Cosmic Voyage 危险："有一个物理定律要尊重"

玩家的敌人：
  - 牛顿（引力不会让你走）
  - 爱因斯坦（光速是极限）
  - 热力学（熵增不可避免）

紧张感来源：
  - 不是"怪物要杀我"
  - 而是"如果我计算错误，引力会把我撕碎"
```

## 二、引力系统

### 2.1 太阳 — 巨型引源

```
位置：太空坐标系原点 (0, 0, 0)
效果：
  - 距离 < 10000：感受到明显引力拉扯
  - 距离 < 5000：需要引擎全功率对抗，否则被吸入
  - 距离 < 3000：隔热层不足的飞船开始过热
  - 距离 < 2000：即使全功率引擎也无法逃脱（事件视界）
  - 距离 < 1000：飞船烧毁，玩家死亡

视觉反馈：
  - 远处：巨大的发光球体，耀斑动画
  - 接近：视野边缘出现扭曲（引力透镜效果）
  - 危险区域：HUD 闪烁警告，飞船震动

生存策略：
  - 安装隔热层 → 可以靠近到 3000
  - 安装辐射盾 → 减少引力对电子设备的影响
  - 利用引力弹弓 → 快速改变航向（高级技巧）

引力弹弓机制：
  - 以特定角度接近太阳
  - 引力加速 → 获得额外速度
  - 在正确时机点火 → 改变方向
  - 节省燃料，但需要精确操控
```

### 2.2 气态巨行星 — 洛希极限

```
木星/土星（纯渲染背景天体，不可降落）

洛希极限概念：
  - 距离太近 → 行星引力差会把飞船撕碎
  - 类似于《星际穿越》中的黑洞场景

效果分层：
  安全距离 (> 8000)：正常飞行，可以看到巨大的球体
  警告区域 (5000-8000)：HUD 警告，飞船轻微震动
  危险区域 (3000-5000)：需要压力壳模块，否则船体损坏
  洛希极限 (< 3000)：即使全功率引擎也无法逃脱，飞船逐渐被撕裂
  死亡 (< 2000)：飞船解体，玩家死亡

视觉反馈：
  - 安全距离：壮观的气态行星充满视野
  - 警告区域：飞船金属发出吱吱声
  - 危险区域：船体部件开始脱落（如果有损坏）
  - 洛希极限：飞船被拉长、撕裂的动画

木星 vs 土星差异：
  - 木星：更强的引力，更大的洛希极限半径
  - 土星：引力较弱，但有环系统（环中的碎片危险）

环系统危险：
  - 土星环 = 大量高速运动的冰块/岩石
  - 穿越环 = 需要护盾/规避
  - 可以在环中采矿（高风险高回报）
```

### 2.3 飞船操控的阻尼感

```
当前设计：太空飞行无阻力（惯性飞行）

引力场内的变化：
  - 接近大质量天体时，操控变得"沉重"
  - 引擎需要更多功率对抗引力
  - 转向更困难（引力在拉偏航向）

数学模型：
  引力加速度 = G * M / r²
  - G = 引力常数（游戏可调）
  - M = 天体质量
  - r = 到天体中心的距离

飞船受力：
  引擎推力 + 引力 = 实际加速度
  
  当引力 >> 引擎推力时：
    → 飞船失控，被拉向天体
    → 需要紧急机动或接受命运

操控体验：
  - 远离天体：轻盈、自由（当前体验）
  - 中等距离：感觉到"拖拽"，需要更多推力
  - 近距离：沉重、紧张，每秒钟都在对抗死亡
  - 太近距离：绝望，看着HUD上的距离数字不断减小
```

## 三、飞船损坏系统

### 3.1 损坏类型

| 损坏类型 | 触发条件 | 效果 | 修复方式 |
|:---|:---|:---|:---|
| **船体裂纹** | 压力壳不足时靠近气态巨行星 | 氧气泄漏，内部失压 | 在外部焊接（EVA） |
| **引擎过热** | 隔热层不足时靠近太阳 | 引擎推力下降，可能熄火 | 冷却等待，或安装散热器 |
| **电子设备故障** | 辐射盾不足时穿越辐射带 | HUD 失灵，自动导航失效 | 重启系统（需要时间） |
| **燃料泄漏** | 碰撞小行星/环碎片 | 燃料快速消耗，无法长途飞行 | 找到安全区域修补 |
| **结构损伤** | 洛希极限撕裂 | 逐渐丧失功能，最终解体 | **无法修复，弃船** |

### 3.2 损坏程度

```
绿区 (100-70%)：正常运作
黄区 (70-30%)：警告，性能下降
红区 (30-10%)：危险，关键系统可能失效
黑区 (<10%)：濒临毁灭，必须立即降落/弃船

可视化：
  - 绿区：飞船外观正常
  - 黄区：部分部件发光/冒烟
  - 红区：火花，船体裂纹可见
  - 黑区：警报声，紧急灯光，部分功能丧失
```

### 3.3 与飞船系统的联动

```
损坏影响系统：
  驾驶舱损坏 → 操控灵敏度下降，HUD信息缺失
  引擎损坏 → 推力下降，最高速度降低
  氧气生成器损坏 → 氧气产生速率下降
  气密外壳损坏 → 氧气泄漏，内部失压

修复方式：
  1. 在前哨站/废弃站完全修复（最快）
  2. 飞船内部维修（需要维修工具+资源）
  3. EVA（舱外活动）焊接（需要气闸门+氧气）
  4. 某些损坏不可逆（结构损伤），需要弃船
```

## 四、生物系统（低优先级）

### 4.1 存在生物的天体

| 天体 | 生物类型 | 生态特征 | 交互方式 |
|:---|:---|:---|:---|
| **木卫二** | 深海生物 | 冰壳下液态海洋，类似地球深海 | 钻探取样，不可直接接触 |
| **土卫六** | 甲烷生态 | 甲烷湖中的微生物聚落 | 采集样本，实验室分析 |

### 4.2 不存在生物的天体

```
水星：极端温差，无大气 → 不可能有生物
金星：硫酸云，表面温度462°C → 不可能有生物
月球：无大气，无水 → 不可能有生物
火星：辐射强，大气稀薄 → 可能曾有微生物（遗迹中可能有化石）

设计原则：
  - 遵循真实科学认知
  - 木卫二和土卫六是现实中科学家认为可能存在生命的地方
  - 其他天体没有 = 保持科学准确性
```

### 4.3 生物的设计原则

```
❌ 不是敌人：不会攻击玩家
❌ 不是资源：不能杀死获取物品
✅ 是叙事元素：通过样本分析了解它们
✅ 是探索奖励：发现生物 = 科学发现

交互方式：
  1. 采集样本（采样臂）
  2. 实验室分析（飞船实验室模块）
  3. 记录发现（日志系统）
  4. 可能的发现：
     - "这些微生物利用化学合成能量，类似地球热泉口的细菌"
     - "它们的外膜可以抵抗极端压力，这启发了新的材料技术"
     - "这些生物群落的分布模式...似乎不是自然形成的？"
```

## 五、危险系统的情感设计

```
Outer Wilds 的危险设计参考：
  - 太阳爆炸：不是"怪物杀你"，而是"宇宙的自然规律"
  - 22分钟循环：给了玩家尝试的勇气（死亡不可怕）
  - 幽灵物质：探索过程中的紧张感

Cosmic Voyage 的情感弧线：
  
  1.  awe（敬畏）
     → 第一次看到巨大的木星充满视野
     → "哇...这太美了"
  
  2.  tension（紧张）
     → 飞船震动，HUD警告，引力在拉扯
     → "我需要调整航向...快点..."
  
  3.  fear（恐惧）
     → 距离数字不断减小，引擎全功率也无法挣脱
     → "完蛋了...我要被撕碎了..."
  
  4.  relief（解脱）
     → 最后一刻点火成功，逃离危险区域
     → "呼...活下来了..."
  
  5.  mastery（掌控）
     → 第二次接近时，成功利用引力弹弓加速
     → "哈哈！我做到了！"

关键：
  - 危险必须可逃避（不能是必死局）
  - 死亡惩罚轻（可以在基地复活，飞船在基地）
  - 每次危险都是学习机会
  - 最终：玩家学会尊重物理，并利用物理
```

## 六、实现优先级

| 优先级 | 功能 | 说明 |
|:---|:---|:---|
| **P0** | 太阳引力效果 | 基础引力场，玩家能感受到拉扯 |
| **P0** | 飞船损坏-基础 | 碰撞小行星 = 船体损伤 |
| **P1** | 气态巨行星洛希极限 | 接近木星/土星的危险区域 |
| **P1** | 飞船损坏-系统联动 | 不同系统损坏有不同效果 |
| **P2** | 引力弹弓机制 | 高级飞行技巧 |
| **P2** | 环系统穿越 | 土星环的碎片危险 |
| **P3** | 生物系统 | 木卫二/土卫六的神秘生物 |
| **P3** | 深海探索 | 木卫二冰壳下的海洋 |
