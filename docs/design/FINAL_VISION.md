## 二、太阳系（8个可降落天体）

| 天体 | 绑定维度 | 类型 | 特征 |
|:---|:---|:---|:---|
| **太阳** | 不可降落 | 恒星 | 发光球体，强引力，靠近烧毁 |
| **水星** | 新维度 | 岩石行星 | 灰白，陨石坑，极端温差 |

| **月球** | 新维度 | 卫星 | 灰白，陨石坑 |
| **火星** | 新维度 | 岩石行星 | 红色，沙尘，古代遗迹 |
| **木卫二** | 新维度 | 卫星 | 冰白，冰裂纹，**深海生物** |

- 金星 → 下界（炎热环境相似）

## 三、核心循环

```
探索 → 发现 → 理解 → 建造

1. 驾驶飞船到达新星球
2. 发现遗迹 → 了解前人故事 + 收集资源
3. 建造/升级飞船 → 获得新能力
4. 新能力 → 到达之前无法到达的地方
5. 到达新地方 → 发现新遗迹 → 循环

知识的作用：
  - 不是"解锁"，是"知道去哪里"
  - "遗迹说火星有钛矿" → 去火星找钛矿 → 造隔热层 → 去水星
  - 信息差 = 进度感
```

## 四、飞船系统

### 4.1 核心哲学

> **飞船不是载具，是家。**
>
> 可建造的移动基地，完全自由设计，科技从一开始完全解锁。

### 4.2 起飞安全检测

必须同时满足：
- ✅ 驾驶舱（飞行控制）
- ✅ 引擎（提供推力）
- ✅ 氧气生成器（生命维持）
- ✅ 气密外壳（船体完整性）

### 4.3 扩展模块（添加式升级）

| 类别 | 模块 | 新能力 |
|:---|:---|:---|
| 环境耐受 | 隔热层 | 靠近太阳/水星 |
| | 辐射盾 | 穿越辐射带 |
| | 压力壳 | 进入气态巨行星大气 |
| | 抗冻层 | 探索冥王星深处 |
| 航程扩展 | 跃迁引擎 | 自动超光速航行，100-1000倍速，消耗氦-3 |
| | 太阳能帆 | 靠近恒星无限续航 |
| | 自动导航 | 自动飞往目标 |
| 探索能力 | 地形扫描仪 | 发现地表遗迹 |
| | 钻探设备 | 深层采矿 |
| 空间扩展 | 货仓扩展 | 更多存储空间 |
| | 生活舱 | 带NPC船员 |
| | 实验室 | 分析外星样本 |

**所有模块从一开始完全可用，无科技锁。**

### 4.4 蓝图系统

遗迹中的前人飞船设计记录：
- 叙事功能：了解前人的故事
- 学习功能：新手参考成熟设计
- 收藏功能：收集所有蓝图作为成就

**不是解锁机制。玩家可以完全无视蓝图，自由设计。**

## 五、危险系统

### 5.1 核心哲学

> **宇宙本身就是敌人。**
>
> 没有太空海盗，没有外星军队。危险来自物理本身。

### 5.2 引力危险

| 天体 | 危险类型 | 效果 |
|:---|:---|:---|
| **太阳** | 巨型引源 | 距离<3000需要隔热层，<2000烧毁 |
| **木星/土星** | 洛希极限 | 距离<3000被引力撕裂 |
| **土星环** | 高速碎片 | 穿越需护盾/规避 |

### 5.3 飞船损坏

| 损坏类型 | 触发条件 | 效果 |
|:---|:---|:---|
| 船体裂纹 | 压力壳不足+气态巨行星 | 氧气泄漏 |
| 引擎过热 | 隔热层不足+太阳 | 推力下降 |
| 电子设备故障 | 辐射盾不足 | HUD失灵 |
| 结构损伤 | 洛希极限 | **不可逆，弃船** |

### 5.4 生物

| 天体 | 生物 | 类型 |
|:---|:---|:---|
| 木卫二 | 深海生物 | 探索叙事 |
| 土卫六 | 甲烷微生物 | 探索叙事 |
| 其他天体 | **无** | 科学准确性 |

**生物不会攻击玩家，纯叙事元素。**

## 六、前哨站系统

### 6.1 遗迹（纯叙事）

- 散落在星球表面的建筑残骸
- 提供背景故事碎片
- 不可修复为功能性建筑

### 6.2 星文系统

- 遗迹中的几何图案
- 玩家逐步学习解读
- 多语言适配（图案是视觉语言）

### 6.3 前哨站（玩家建造）

**默认极简模式**（5分钟建站）：
- 放置基础框架 → 自动生成墙壁
- 放置功能方块 → 自动连接
- 氧气：室内自动有
- 电力：太阳能板范围内自动供电

**功能方块**：
- 气闸门、氧气生成器、太阳能板
- 飞船对接端口、储物舱、观测窗

### 6.4 废弃站（混合）

- 世界自动生成，半损坏
- 资源即时修复 → 功能启用
- 可在此基础上扩建

### 6.5 联动接口

```
本体：轻量但完整的前哨站系统
联动 Create：机械动力、管道系统、自动化
联动 MineColonies：NPC宇航员、殖民地管理

"本体提供完整但轻量的体验
 联动提供深度但可选的体验
 玩家选择自己的复杂度"
```

## 八、航行系统：巡航模式 + 跃迁引擎

### 8.1 巡航模式（短距离/手动）

- Shift+W 激活，速度提升10-50倍
- 转向大幅减弱（只能微调）
- HUD显示航线预测、到达时间
- 任意方向键或鼠标大幅移动 → 退出
- **用途**：小行星带机动、靠近行星、空间站对接、看风景

### 8.2 跃迁引擎（长距离/自动）

> **跃迁引擎不是瞬间传送，是自动超光速航行。**
> 解决"行星间距离太远，手动飞几十分钟纯枯燥"的问题。

**操作流程**：
1. 打开HUD星图 → 选择一个已知天体（已探索过的）
2. 引擎自动转向对准目标
3. 启动跃迁 → 速度提升100-1000倍，星空变成流线效果
4. 到达目标天体引力范围 → 自动退出 → 减速至正常飞行

**燃料**：消耗氦-3（月球特产），消耗量 = 距离 × 飞船质量
- 轻量船壳优势：同样燃料跃迁更远
- 大飞船载货多但跃迁成本高 → 有趣的权衡决策

**风险**：
- 航线上有小行星/碎片 → 跃迁中断 → 紧急退出
- 燃料中途耗尽 → 漂流太空 → 需救援
- 误跃迁至气态巨行星附近 → 洛希极限危险

**与巡航模式的对比**：

| | 巡航模式 | 跃迁引擎 |
|:---|:---|:---|
| 操控 | 手动 | 自动导航 |
| 速度 | 10-50× | 100-1000× |
| 用途 | 短距离/机动/探索 | 行星间旅行 |
| 风险 | 可手动避障 | 航线障碍会中断 |
| 消耗 | 低燃料 | 高燃料（氦-3） |

## 十二、上古虫洞与传送门（未来拓展科技线）

### 12.1 问题

原版 Minecraft 的地狱传送门和末地传送门在我们的模组中依然存在（金星绑定下界、冥王星绑定末地）。但在一个追求科学一致性的太空探索模组中，"用打火石点燃黑曜石框架就能打开异次元"显得格格不入。

### 12.2 解法：上古文明的微型虫洞网络

> **传送门不是魔法，是上古文明遗留的微型虫洞稳定器。**

**设定**：
- 太阳系曾经存在一个高度发达的文明（即"前人"）
- 他们在行星间建立了稳定的微型虫洞网络用于快速航行
- 虫洞终端留在金星（下界）和冥王星（末地）
- 传送门的黑曜石框架是虫洞稳定器的"外壳残骸"，打火石只是重新激活了仍在低功耗运行的设备
- 下界的"地狱"外观 = 虫洞另一端连接了一个高温废弃世界（可能是前人改造的工业星球）
- 末地的"虚空"外观 = 虫洞另一端连接了一个被撕裂的破碎世界

### 12.3 与现有设定的融合

| 原版元素 | 虫洞解释 |
|:---|:---|
| 黑曜石框架 | 虫洞稳定器的外壳残骸 |
| 打火石激活 | 重新启动低功耗运行的设备 |
| 瞬间传送 | 微型虫洞折叠空间 |
| 下界（金星绑定） | 前人改造的高温工业世界 |
| 末地（冥王星绑定） | 被引力撕裂的破碎世界 |
| 末影龙 | 前人遗留的自动防御系统 |
| 下界要塞 | 前人前哨站的废墟 |

### 12.4 未来拓展方向

这条科技线留给 Phase 3+ 实现，不急于现在开发：

**虫洞科技线**：
- 发现前人虫洞技术的原理 → 学习"虫洞工程学"
- 收集前人遗迹中的虫洞核心碎片
- 最终目标：**建造自己的虫洞网络**
- 可以在任意两个前哨站之间建立稳定的虫洞通道
- 甚至可以在行星轨道上建立"虫洞门"——飞船飞进去，从另一个行星轨道出来

**与跃迁引擎的区别**：

| | 跃迁引擎 | 虫洞网络 |
|:---|:---|:---|
| 速度 | 近光速（100-1000×） | 瞬间（加载时间） |
| 能耗 | 氦-3，按距离计算 | 启动成本高，维护成本低 |
| 基建 | 不需要 | 两端都需要虫洞门 |
| 限制 | 飞船必须飞过去 | 只有建了门的地点才能互通 |
| 用途 | 探索、首次到达 | 已建立殖民地的贸易路线 |

**这自然地从"探索科技"过渡到"文明建设科技"**：
- Phase 1-2：跃迁引擎探索太阳系
- Phase 3+：建立殖民地 → 虫洞网络连接 → 星际贸易/帝国

### 12.5 叙事价值

这个设定还解决了原版 Minecraft 叙事的空缺：
- "为什么地狱和末地存在？" → 上古文明的虫洞网络
- "前人去了哪里？" → 也许是虫洞网络崩溃时被困在了另一端
- "玩家的探索有什么意义？" → 重建前人失去的技术，避免同样的命运

## 设计哲学检查清单

| 哲学 | 应用 |
|:---|:---|
| **真实** | 太阳系布局、引力物理、生物分布、温度/气压考量 |
| **孤独** | 一个人的宇宙，前人的遗迹，荒凉的环境 |
| **探索驱动** | 知识即进度，不是等级/装备 |
| **添加式升级** | 新能力而非数值+10 |
| **完全自由** | 科技从一开始解锁，无 grinding |
| **分层复杂度** | 本体轻量，联动深度，玩家自选 |
| **视听体验** | LOD渲染、巨物感、HUD导航、巡航模式 |
| **科幻一致性** | 传送门=上古虫洞，跃迁=近光速航行 |
