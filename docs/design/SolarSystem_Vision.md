## 二、原版维度绑定设计

### 金星 ↔ 下界
```
相似性：
  - 下界：红色天空、岩浆、炎热
  - 金星：橙黄大气、表面温度462°C、硫酸云

绑定方式：
  - 太空维度中看到一个橙黄色星球（金星渲染）
  - 降落后进入的是 下界维度（原版地形+结构）
  - 但重力、天空盒、物理参数不同（金星参数覆盖）
  
游戏逻辑：
  - 玩家知道"这是金星"
  - 但地形是下界（熟悉但新奇）
  - 可以解释："金星的地下世界与下界相连"
```

### 冥王星 ↔ 末地
```
相似性：
  - 末地：紫色天空、浮空岛、荒凉
  - 冥王星：极寒、紫灰色调、神秘

绑定方式：
  - 太空维度中看到一个紫灰色矮行星
  - 降落后进入 末地维度（原版地形）
  - 物理参数覆盖：极低温、低重力
  
游戏逻辑：
  - "冥王星的表面与末地维度相连"
  - 末影龙可以解释为"冥王星的守护者"
```

### 不绑定的行星
```
水星、地球、月球、火星、木卫二、土卫六 = 新建维度
  - 全新的地形生成器
  - 独特的方块和生物
  - 独立的物理参数（重力、温度等）
```

## 三、太空坐标系布局

```
太阳(0,0,0)
  |
  |-- 水星 @ 4000
  |-- 金星 @ 7000
  |-- 地球 @ 10000
  |     \-- 月球 @ +2000 (相对地球)
  |-- 火星 @ 15000
  |-- (小行星带) @ 18000-22000
  |-- 木星轨道区域 @ 25000 (木卫二)
  |-- 土星轨道区域 @ 35000 (土卫六)
  |-- (天王星/海王星可选)
  \-- 冥王星 @ 50000
```

### 距离设计原则
```
相邻行星间距：3000-5000
飞船 MAX_SPEED_TICK = 25 blocks/tick
实际速度：约 25 * 20 = 500 blocks/second

地球→月球：2000 / 500 = 4秒（短程）
地球→火星：5000 / 500 = 10秒（中程）
地球→冥王星：40000 / 500 = 80秒（长途，需要Warp或跃迁)
```

## 四、渲染形态构想（美术方向）

### 太阳
```
渲染方式：纯发光球体，非实体
- 半径：远大于行星（EARTH_RADIUS * 5 = 1000）
- 效果：辉光 + 日冕粒子
- 危险：靠近会被烧毁
- 作用：方向光源（所有行星的照明来源）
```

### 气态巨行星（纯渲染背景，不可降落）
```
木星：条纹状纹理，大红斑
土星：环系统（需要环形渲染技术）

注：不计划可降落，只作为远处可见的背景天体
增加太空的壮观感和尺度感
```

### 行星渲染 LOD 扩展
```
当前 LOD 三级：Billboard → 低模球体 → 完整球体+大气

扩展：
  LOD_FAR (8000+)     → Billboard 光点（远距离行星）
  LOD_MID (2000-8000) → 低模球体
  LOD_NEAR (1000-2000) → 完整球体+大气
  LOD_DETAIL (<500)    → 表面纹理可见（需要新 shader）
  LAND (<TRIGGER)      → 维度切换
```

## 七、设计哲学（已确认）

> **"真实才能打动人"**

### 已确认决策

| # | 决策 | 理由 |
|:---|:---|:---|
| 1 | **8个可降落天体，数量不变** | 刚好覆盖内太阳系+外太阳系标志性天体 |
| 2 | **气态巨行星不可降落，作为巨物感来源** | 真实的巨行星没有固体表面，保持科学准确性 |
| 3 | **巡航模式加速长距离旅行** | 保留手动驾驶参与感 + 长距离自动巡航效率 |
| 4 | **小行星带暂时搁置** | 需要实体物理系统（运动+交互），实现难度过高；纯渲染贴图无游戏性 |

### 设计哲学应用

**气态巨行星的正确用法**：
```
不是"降落到表面"，而是"在大气层内部飞行"

体验设计：
  1. 远处：Billboard 光点（木星条纹/土星环可见）
  2. 接近：巨大的球体充满视野（巨物感核心来源）
  3. 大气层边缘：进入彩色云层飞行（不需要实体地形）
  4. 内部深处：压力和温度致命，无法继续深入
  
不需要：地形生成、方块放置、生物生成
只需要：大气层 Shader + 粒子效果 + 危险区域检测
```

**巡航模式设计**：
```
手动模式（当前）：WASD + 鼠标，精细操控，用于着陆/近距离
巡航模式（新增）：
  - 激活方式：Shift + W 或 HUD 按钮
  - 速度：手动模式的 10-50 倍
  - 转向：大幅度减弱（只能微调方向，不能急转）
  - HUD：显示航线预测、到达时间
  - 退出：任意方向键或鼠标大幅移动
  
Outer Wilds 参考：
  - Marshmallow（自动对齐速度）
  - Auto-pilot（自动导航到目标）
```

**小行星带后续条件**：
```
何时加入：
  - 需要实体方块运动系统（每个小行星是实体，有轨道运动）
  - 或：实例化渲染（Instanced Rendering）技术成熟
  - 或：简化方案（小行星 = 静态结构生成，不运动）
  
当前替代方案：
  - 小行星带区域 = 高风险的陨石雨（粒子效果 + 碰撞伤害）
  - 资源采集 = 月球/火星表面的矿石（静态方块）
```
