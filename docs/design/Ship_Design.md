# 飞船设计文档 — 可建造的移动基地

> 设计哲学：**飞船不是载具，是家**
>
> 玩家建造自己的飞船，带着它探索宇宙。飞船是可扩展的移动基地，每次升级 = 新能力，不是数值+10。

---

## 一、核心概念

### 飞船 = 可建造实体

```
类似 Create 的列车/机械，但设计更简单：
  - 有内部空间（类似船的内部）
  - 玩家可以在内部走动、放置方块
  - 核心系统方块必须放置才能起飞
  - 扩展模块添加新能力
```

### 起飞安全检测

```
必须同时满足：
  ✓ 驾驶舱（飞行控制）
  ✓ 引擎（提供推力）
  ✓ 氧气生成器（生命维持）
  ✓ 气密外壳（船体完整性）

不满足时：
  - 无法启动引擎
  - HUD 提示缺少什么："缺少氧气生成器 — 无法起飞"
  - 可以在地面滑动/拖拽（无动力移动），但不能飞行
```

### 添加式升级

```
❌ 不是："A矿 x10 → 引擎速度 +10"
✅ 而是："安装隔热层 → 可以靠近太阳"

每次升级都是新能力：
  - 新区域：可以探索之前无法到达的地方
  - 新功能：可以做之前做不到的事情
  - 新空间：可以在飞船上做更多事情
```

## 二、核心系统（必须）

| 系统 | 功能 | 不安装的后果 |
|:---|:---|:---|
| **驾驶舱** | 飞行控制、HUD显示 | 无法控制飞船 |
| **引擎** | 提供推力 | 无法飞行 |
| **氧气生成器** | 维持内部氧气 | 内部无氧气，玩家窒息 |
| **气密外壳** | 船体完整性 | 无法进入太空维度 |

### 气密外壳检测

```
简化设计（不需要 Space Engineers 级别的复杂检测）：
  - 飞船由"船体方块"围成封闭空间
  - 检测：所有驾驶舱/氧气生成器是否被船体方块包围
  - 漏洞 = 任何方向通向外界的缺口
  - 修复：填补缺口即可
  
可视化：
  - 安装"气密检测器"方块
  - 绿色 = 气密良好
  - 红色 = 检测到泄漏（显示泄漏位置）
```

## 三、扩展模块（完全开放，无科技锁）

> **设计变更**：所有模块从一开始完全可用，遗迹只提供叙事和前人的设计参考。
> 玩家可以完全自由地设计自己的飞船，不受任何解锁限制。

### 环境耐受类

| 模块 | 新能力 | 效果 |
|:---|:---|:---|
| **隔热层** | 耐受极端高温 | 可以靠近太阳/水星 |
| **辐射盾** | 阻挡宇宙辐射 | 可以穿越辐射带 |
| **压力壳** | 耐受高压环境 | 可以进入气态巨行星大气 |
| **抗冻层** | 耐受极低温 | 可以探索冥王星深处 |

### 航程扩展类

| 模块 | 新能力 | 效果 |
|:---|:---|:---|
| **额外燃料罐** | 增加燃料储备 | 飞得更远，不需要中途加油 |
| **太阳能帆** | 恒星能源收集 | 靠近恒星时无限续航 |
| **跃迁引擎** | 长距离传送 | 可以瞬间跳跃到远距离行星 |
| **自动导航** | 自动飞往目标 | 设定目标后自动飞行 |

### 探索能力类

| 模块 | 新能力 | 效果 |
|:---|:---|:---|
| **地形扫描仪** | 发现地表遗迹 | 降落前显示地表兴趣点 |
| **钻探设备** | 深层采矿 | 采集普通挖矿无法获得的矿物 |
| **采样臂** | 收集外星样本 | 带回实验室分析 |
| **信号接收器** | 接收遗迹信号 | 指引玩家找到附近的遗迹 |

### 空间扩展类

| 模块 | 新能力 | 效果 |
|:---|:---|:---|
| **货仓扩展** | 增加存储空间 | 带更多资源往返 |
| **生活舱** | 可以带NPC船员 | NPC帮助维护/研究 |
| **实验室** | 分析外星样本 | 分析样本 |
| **观测穹顶** | 更好的视野 | 360度观察太空，情绪价值 |

## 四、蓝图系统 — 前人的设计遗产

> **所有模块从一开始完全可用。蓝图不是解锁机制，是前人的飞船设计方案。**

### 蓝图是什么

```
前人在遗迹中留下的飞船设计记录：
  - "第7探险队的侦察船：3x3x5，核心系统+隔热层+扫描仪"
  - "火星前哨站的货运船：5x5x8，大货仓+太阳能帆+自动导航"
  - "远征队的旗舰：7x7x12，全模块搭载"

玩家可以：
  ✓ 完全无视蓝图，自己从零设计
  ✓ 参考蓝图学习布局思路（"原来引擎放在重心后面更稳定"）
  ✓ 复刻前人的飞船（体验历史感）
  ✓ 在前人基础上改进（"这个设计不错，但我要加跃迁引擎"）

蓝图的作用：
  - 叙事：了解前人的故事（"这支探险队在金星失踪了..."）
  - 学习：新手可以参考成熟设计
  - 收藏：收集所有蓝图作为成就
```

### 模块获取方式

```
所有模块：直接合成（资源收集 → 合成台 → 获得）
没有任何科技锁，没有任何前置条件

唯一限制：资源
  - 某些高级模块需要稀有资源
  - 稀有资源只在特定星球出现
  - 但这只是"在哪里找"的问题，不是"能不能用"的问题

遗迹的作用（纯叙事）：
  - 遗迹中的日志提到"我们在火星发现了钛矿石，用它造了隔热层"
  - 玩家："哦，火星有钛矿，可以去那里造隔热层"
  - 不是解锁，是提示
```

## 五、飞船尺寸与复杂度

```
小型侦察船（推荐新手）
  - 3x3x5 内部空间
  - 核心系统 + 1-2个扩展模块
  - 灵活快速，适合近距离探索

中型探索船（推荐标准）
  - 5x5x8 内部空间
  - 核心系统 + 3-5个扩展模块
  - 平衡的速度和功能

大型远征船（后期）
  - 7x7x12 内部空间
  - 核心系统 + 6+个扩展模块
  - 可以长期远征，但需要更多资源维护

限制：
  - 飞船不能无限大（性能考虑）
  - 更大的飞船 = 更慢的速度
  - 玩家需要根据任务选择合适的飞船
```

## 六、情感设计

```
"家"的感觉：
  - 飞船是玩家在荒凉宇宙中唯一安全的地方
  - 玩家自己建造、装饰、扩展
  - 每个飞船都是独一无二的

成长感：
  - 初始：简陋的小舱，只能飞到月球
  - 中期：装备了扫描仪和跃迁引擎，探索多个行星
  - 后期：大型远征船，可以探索太阳系的每个角落

孤独与温暖：
  - 外部：荒凉、寒冷、致命的太空
  - 内部：温暖的灯光、熟悉的布局、安全的氧气
  - 对比 = 情绪冲击

Outer Wilds 参考：
  - OW的飞船虽小但充满细节（烤棉花糖、植物、乐器）
  - CV的飞船由玩家建造，但同样需要"生活气息"
  - 建议：可以放置装饰性方块（照片、植物、纪念品）
```

## 七、实现考虑

```
技术挑战：
  1. 可建造实体 = 需要自定义实体类型
  2. 内部空间 = 需要子维度或虚拟空间
  3. 气密检测 = 需要区域检测算法
  4. 多艘飞船 = 需要飞船管理UI

可能的简化方案：
  - V1：飞船 = 固定尺寸的多方块结构（类似铁傀儡的召唤方式）
  - V2：真正的可建造飞船（类似 Create 的列车）
  - V3：完全自由的飞船建造（Space Engineers 级别）

建议：V1 足够支撑 Phase 2 内容，V2/V3 留给 Phase 3
```

## 八、与核心循环的整合

```
探索循环：
  1. 建造飞船（完全自由，任何模块都可以装）
  2. 驾驶飞船到达新星球
  3. 发现遗迹 → 了解前人故事 + 找到资源提示
  4. 发现新资源 → 造更多模块 → 改进飞船
  5. 到达新地方 → 发现新遗迹 → 循环

知识的作用（Outer Wilds 式）：
  - 不是"解锁"，是"知道去哪里"
  - "遗迹说火星有钛矿" → 去火星找钛矿 → 造隔热层 → 去水星
  - 信息差 = 进度感，不是科技锁 = 进度感

玩家的飞船演变：
  - 新手：参考蓝图，造一艘简单的侦察船
  - 中期：自己设计，针对特定任务优化
  - 后期：拥有多艘飞船，每艘针对不同任务

核心乐趣：
  - 设计自己的飞船（"这是我的船"）
  - 探索未知（"那个星球上有什么？"）
  - 发现故事（"前人发生了什么？"）
```
