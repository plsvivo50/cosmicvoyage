## 一、今日完成事项

### 1.1 编译错误修复（紧急）

另一个 IDE AI 助手引入了大量回归 bug，今日首要任务是回退并修复。

| 文件 | 问题 | 修复 |

### 1.2 飞船控制修复（P0）

| 修复 | 说明 |
|:---|:---|
| 六方向正常飞行 | W/S 前后、A/D 偏航转弯、Space 上升、LeftCtrl 下降 |
| Space 跳跃冲突 | 每 tick 重置 `mc.player.input.jumping = false`，阻止原版骑乘跳跃逻辑 |

| 删除自动对齐/着陆代码 | 移除 `targetPitch`、`alignPitchToVelocity` 等未就绪的复杂逻辑，保持简单 |

### 1.3 碰撞反馈（Issue #2）

| 修复 | 说明 |
|:---|:---|
| 撞墙速度衰减 | `move()` 后检测 `horizontalCollision`，速度 `scale(0.3)`，快速停止 |
| 结果 | 全速撞墙后不再远距离滑行，物理感改善 |

### 1.4 地球尺寸放大（Issue #3）

| 修改 | 说明 |
|:---|:---|
| `EARTH_RENDER_RADIUS` | 50.0f → 200.0f（4倍） |
| 结果 | 近距离巨物感显著增强 |

### 1.5 远距离天体 2D HUD 指示器（Issue #6）

**新增文件**：`NavigationHUDRenderer.java`

| 功能 | 说明 |
|:---|:---|
| 屏幕空间投影 | 目标在视野中央时 HUD 在屏幕中央，目标偏离时 HUD 退到边缘 |
| 三段式 LOD | 近距离(<600格)隐藏让3D球体渲染；中距离屏幕内色块点+距离；远距离屏幕边缘箭头+距离 |
| 透视正确 | 抬头/低头/左转/右转均正确映射到屏幕坐标 |
| 方向箭头 | 天体在视野外时，箭头指向天体方向，文字朝屏幕中心偏移 |

**调试历程**：透视投影 Y 轴翻转 → 水平方向镜像 → atan2 角度环绕导致文字跳变 → 最终用角度差 + 屏幕空间归一化方案收敛。

## 三、技术决策记录

| 决策 | 选择 | 理由 |
|:---|:---|:---|
| HUD 投影方案 | 角度差 + 屏幕空间归一化 | 比 view-space 矩阵投影更稳定，避免坐标系定义错误 |
| 虚空伤害处理 | 暂不修复，记为 #9 | `the_end` 效果硬编码，需替换整个 DimensionSpecialEffects，复杂度超出今日范围 |
| 视角控制 | 保持简单六自由度 | Outer Wilds 式解耦需要重新设计输入系统，后续统一迭代 |
| 自动着陆/过渡 | 移除复杂状态机 | 当前只需要 `/cvspace moon` 基础切换，后续做过场动画时再增强 |
