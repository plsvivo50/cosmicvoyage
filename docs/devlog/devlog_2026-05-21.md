### 关键决策记录

**火星视觉参数**（最终确认）：
- 半径 100 = 1/2 地球，比月球(50)大一倍，体现行星尺度差异
- 颜色 `{0.65, 0.25, 0.12}` 铁锈红，大气 `{0.80, 0.45, 0.30}` 淡橙红
- 地形：red_sandstone(8层) + coarse_dirt(2层)，与月球黑曜石区分

**坐标布局**（最终确认）：
- 太阳 `(0, 0, 0)` 纯渲染背景
- 地球 `(10000, 0, 0)`
- 月球 `(12000, 0, 0)` 距地球 +2000
- 火星 `(15000, 0, 0)` 距地球 +5000

**ClassLoading 机制**（新认知）：
- `@Mod.EventBusSubscriber` 依赖 JVM 类加载才能生效
- MarsRenderer 最初没有任何代码引用 → 类不加载 → 事件不注册 → 渲染不触发
- 通过 NavigationHUDRenderer 引用 `MarsRenderer.MARS_POSITION` 确保类加载
- EarthRenderer/MoonRenderer 能工作是因为 HUD 早就引用了它们的 POSITION 常量
