## 六、核心架构决策记录

| 决策 | 选择 | 理由 |
|:---|:---|:---|
| 坐标精度方案 | camera-relative 渲染 | VT-1 验证通过，百万格稳定 |
| 飞船同步方案 | 客户端权威 + 自定义网络包 | `isControlledByLocalInstance()` + `ShipSyncPacket` |
| 碰撞检测 | 原版 `move()` AABB | VT-3B 验证通过，25格/tick撞1格墙不穿模 |
| 网络通道 | 独立 `SimpleChannel` | 绕过原版载具同步，零回跳 |
| 星空渲染 | 预生成 + DEBUG_LINES | 内存稳定，2000颗星无压力 |
