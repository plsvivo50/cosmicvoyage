## 一、今日完成事项

### 1.1 六自由度飞行物理重构（P0）

**改动文件**：`ShipEntity.java`、`ShipInputHandler.java`

| 功能 | 说明 |
|:---|:---|
| **本地坐标系** | 基于 `forward × worldUp = right`、`right × forward = up` 构建完整的 6DoF 正交基 |
| **Outer Wilds 式控制** | 鼠标控制飞船 pitch + yaw，A/D 改为左右平移（strafe），W/S 沿机头前后 |
| **纯净 DeltaMovement** | `clientTickPhysics()` 只计算推力+速度+`setDeltaMovement()`，`tick()` 统一执行 `move()` |
| **视角跳变修复** | 删除 `xRotO/yRotO` 显式覆盖，由原版 `Entity.tick()` 自动管理渲染插值 |
| **Pitch 方向修正** | `mouseY - lastMouseY`（前推鼠标 = 低头，后拉 = 抬头，符合直觉） |
| **本地 up 推力** | Space/Ctrl 沿 `localUp`（飞船头顶/脚底）而非世界 Y 轴，垂直飞行时 W 与 Space 不再冲突 |
| **万向节锁临时防御 B** | pitch ≈ ±90° 退化时继承上一帧 `lastRight`（详见 Issue #11） |

**输入映射**：
```
W/S      → 沿机头前后（forward 方向）
A/D      → 左右平移（right 方向）
Space    → 沿飞船头顶上升（localUp 方向）
LeftCtrl → 沿飞船脚底下降（-localUp 方向）
鼠标移动 → 控制飞船俯仰 + 偏航（乘客视角同步跟随）
```

### 1.2 FOV 缩放问题修复（P0）

| 问题 | 根因 | 解决历程 | 最终方案 |
|:---|:---|:---|:---|
| 按住 Space 时视角一缩一缩 | 原版空格键触发疾跑判定，FOV 周期性缩放 | sprinting 设 false 两次失败（时序问题） | `ComputeFovModifierEvent` 直接锁定 FOV = 1.0f |

### 1.3 虚空伤害修复（P0）

| 问题 | 根因 | 解决历程 | 最终方案 |
|:---|:---|:---|:---|
| 太空/月球掉出世界边界受伤 | `minecraft:the_end` 效果硬编码伤害 | DimensionSpecialEffects 替换（纯客户端，无效）→ LivingHurtEvent 残留特效 → LivingAttackEvent 无特效但实体仍被销毁 | **三重方案叠加**：自定义 `cosmicvoyage:space` effects + `LivingAttackEvent` 拦截 + dimension_type 高度扩大（min_y=-2032, height=4064） |

## 三、失败与教训

| 问题 | 尝试 | 失败原因 | 教训 |
|:---|:---|:---|:---|
| FOV 修复 | sprinting 设 false（两次） | Forge 1.20.1 时序复杂，sprinting 被多处重置 | 事件驱动的方案比状态补丁更可靠 |
| 虚空实体销毁 | 覆盖 `checkOutOfWorld()` | 1.20.1 `Entity` 类中该方法不存在，逻辑硬编码在 `tick()` | 先查清楚方法是否存在再写代码 |
| 虚空实体销毁 | `tick()` 中临时调整 y | `setPos()` 触发位置同步，网络层可能干扰 | 不要 hack tick 时序，改配置更干净 |
| 虚空伤害 | `DimensionSpecialEffects` 替换 | 纯客户端类，不影响服务端伤害判定 | 客户端渲染和服务端逻辑分开考虑 |

## 四、技术决策记录

| 决策 | 选择 | 理由 |
|:---|:---|:---|
| 姿态表示 | 欧拉角 Pitch/Yaw（临时） | 简单直接，95% 场景正常；±90° 退化由 lastRight 兜底 |
| FOV 修复 | `ComputeFovModifierEvent` 硬覆盖 | 根治式方案，不依赖对原版 sprinting 机制的理解 |
| 坐标系 | 纯本地（forward/right/localUp） | 所有方向统一参考系，无世界 Y 轴特例 |
| 鼠标-飞船耦合 | 完全同步（视角 = 飞船姿态） | Outer Wilds 式沉浸感，Pitch 控制后着陆姿态可自然对齐 |
| 虚空伤害 | LivingAttackEvent + 高度扩大 | 事件拦截无特效残留 + JSON 扩大根除实体销毁 |
